# Guia de entrega EV03

Esta carpeta ya contiene los elementos principales que solicita la evidencia:

- archivos del proyecto en Java Web con Servlets y JSP
- servicios web para listar, registrar y eliminar citas
- servicio web de autenticacion para administrador
- documentacion de endpoints en `README.md`, `PRUEBAS_API.md` y `docs/`
- archivo `ENLACE_REPOSITORIO.txt` con la URL del repositorio GitHub

## Nombre sugerido del archivo comprimido

Debes cambiar el nombre final del comprimido usando tu informacion personal con este formato:

```text
NOMBRE_APELLIDO_AA5_EV03.zip
```

Ejemplo:

```text
LUIS_BOHORQUEZ_AA5_EV03.zip
```

## Contenido sugerido del comprimido

- carpeta completa del proyecto
- `ENLACE_REPOSITORIO.txt`
- documentacion incluida en `docs/`

## Sustentacion breve sugerida

Puedes explicar que el proyecto fue desarrollado con Java, Maven, Servlets y JSP. La aplicacion ofrece una interfaz web para registrar y administrar citas, y adicionalmente expone servicios web en formato JSON para integracion o pruebas con herramientas como Postman. El repositorio fue manejado con Git y GitHub como evidencia de versionamiento.
